# ⚡ EnerQuiz

**Software Engineering Project (PES)**

EnerQuiz is a trivia game focused on energy consumption in Catalonia, designed to raise awareness and educate users in an interactive way.

The data used in this project has been obtained from the Open Data service of the Government of Catalonia.

**More info:** -> [Project Report](./20240112-EnerQuiz-memoria.pdf)

---

## 🚀 Technologies Used

**Frameworks**
- React Native (Frontend)
- Laravel (Backend)

**Database**
- MySQL
